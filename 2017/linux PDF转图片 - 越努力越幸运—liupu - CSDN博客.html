linux PDF转图片

最近有将PPT首页做成封面的需求，可以通过PPT—>PDF—>图片的方式来获得，这里记录下PDF转图片的方法。

（PPT及PPTX转PDF：点击打开链接）

通过ImageMagick包可以实现PDF转图片

# yum install ImageMagick

安装好后通过命令：

# convert pdf_name.pdf image_name.jpg      //可指定路径

然后就会生成 image_name-0.jpg,image_name-1.jpg等等图片，数目由PDF页数决定

由于只需要转化第一页，可以通过下面的命令来指定页码：

# convert pdf_name.pdf[0] image_name.jpg
可能需要指定生成图片的像素，质量等参数，可以通过以下命令： 

# convert -resize 1800x -density 150 -quality 100 pdf_name.pdf image_name.jpg
-resize 1800x 指定生成的像素大小，越大生成的图片越大，转化的时间越久

-density 150 参数指定密度，具体含义再查

-quality 100 指定生成图片的质量
